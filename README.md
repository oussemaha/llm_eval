# TanitAI Assistant

A multimodal AI assistant capable of handling text, audio, and image inputs.

## Features
- **Text Chat:** Chat with a powerful LLM.
- **Voice Input:** Transcribe and process audio recordings using OpenAI Whisper.
- **Image & Document Processing:** Upload images or PDFs for visual analysis and extraction.
- **Web Search & Retrieval:** Uses custom tools to retrieve and process information.
- **Modern UI:** Built with a sleek Gradio interface.

## Setup & Installation

> [!NOTE]
> A `requirements.txt` file will be added soon!

1. Clone this repository.
2. (Soon) Install dependencies via `pip install -r requirements.txt`.
3. Set up your environment variables (see `template.env`).

## Running the App

Start the Gradio web interface by running:
```bash
python3 main.py
```
